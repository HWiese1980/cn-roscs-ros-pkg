RosDebug Method

Wraps a version of ROS_DEBUG.

Syntax

public void RosDebug (string str)

Parameters

str
The debug message.

Remarks

Currently, this call maps to ROS_DEBUG_THROTTLE(5,"%s",str). This means that debug messages from this node are only sent every 5sec, since ROS identifies debug calls by line number.

Syntax

Parameters

topic
The name of the topic.
callback
The callback method to be called upon reception of a message. The type of this callback determines the type of the message expected.

Remarks

In the generated code, this method is overloaded for every type of message available.

UnSubscribe Method

Parameters

callback
